Web 开发学习笔记

1. 静态网站与动态网站的区别

静态网站：

  • 页面内容固定不变，通常由 HTML、CSS 和 JavaScript 文件组成。
  • 服务器直接传输预先构建好的文件，无需动态生成或数据库查询。
  • 加载速度快，适用于内容更新不频繁的展示型网站。

动态网站：

  • 页面内容根据用户请求或其他条件动态生成。
  • 使用服务器端编程语言（如 PHP、Java、Python 等）与数据库结合。
  • 具有更高的交互性和灵活性，适用于需要频繁更新或用户交互的网站。

2. CSS 样式表

CSS（层叠样式表）：

  • 用于描述 HTML 或 XML 文档的样式。
  • 控制网页的布局、颜色、字体等视觉效果。
  • 可以与脚本语言配合，动态地对网页元素进行格式化。

3. JavaScript 脚本

JavaScript：

  • 一种脚本编程语言，用于为网页添加交互性和动态效果。
  • 可以实现实时内容更新、交互式地图、动画等复杂功能。
  • 是 Web 技术的核心组成部分之一。

4. HTML、CSS、JavaScript 和 Python 的关系

HTML（超文本标记语言）：

  • 用于描述网页的结构和内容。
  • 通过标签定义文本、图像、链接等元素的布局。

CSS（层叠样式表）：

  • 用于控制网页的样式和布局。
  • 定义颜色、字体、间距等视觉效果。

JavaScript：

  • 用于为网页添加交互性和动态效果。
  • 响应用户操作，实现动态更新和交互功能。

Python：

  • 一种高级通用编程语言，用于后端开发。
  • 广泛应用于 Web 开发、数据分析、人工智能等领域。

总结：

  • HTML、CSS 和 JavaScript 主要用于前端开发。
  • Python 通常用于后端开发。
  • 它们共同构成完整的 Web 开发技术栈。

5. 样式表与脚本的区别

样式表：

  • 用于定义网页元素的外观和布局。
  • 例如：CSS 控制颜色、字体、间距等。

脚本：

  • 用于为网页添加动态功能和交互性。
  • 例如：JavaScript 实现表单验证、动态加载内容等。

总结：

  • 样式表负责网页的外观设计。
  • 脚本负责网页的动态功能和交互。
